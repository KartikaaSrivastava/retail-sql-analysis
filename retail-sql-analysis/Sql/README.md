# Retail Sales Analysis using SQL

## Overview
This project demonstrates an end-to-end **retail sales analysis** using SQL on a real-world styled dataset.  
The objective is to uncover **revenue trends, customer behavior and promotion effectiveness**, while handling common data quality issues.

---

## Objectives
- Analyze **monthly and seasonal revenue trends**
- Identify **top-performing categories and products**
- Evaluate **profitability across product categories**
- Assess **discount and promotion effectiveness**
- Apply **data cleaning and standardization** techniques

---

## Dataset
- Source: data.world (Retail Sales Dummy Dataset)
- Time Period: 2020–2024
- Includes transactional, product, customer, inventory, and promotion data

---

## Key Insights
- Revenue shows strong seasonality, peaking in **March–April** and **October–November**.
- **Beauty and Clothing** are the primary revenue drivers.
- **Clothing** delivers the highest profitability.
- Discounts provide limited uplift, indicating the need for targeted promotions.

---

## How to Run
1. Import the dataset into DB Browser for SQLite  
2. Execute SQL scripts from the `sql/` folder  
3. Review insights in `insights/business_insights.md`
